# Contributing to Aavya Ethnic Couture

Thank you for your interest in contributing to our women's textile e-commerce platform!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/ecommerce.git`
3. Install dependencies in both frontend and backend directories
4. Create `.env` files from `.env.example` templates
5. Start the PostgreSQL database
6. Run database seeds
7. Start both frontend and backend servers

## Code Style

- Use TypeScript for type safety
- Follow the existing code structure and naming conventions
- Use meaningful variable and function names
- Keep components small and focused
- Write descriptive commit messages

## Commit Messages

Follow this format:
- `feat: add new feature`
- `fix: resolve bug`
- `docs: update documentation`
- `style: formatting changes`
- `refactor: code restructuring`
- `test: add or update tests`

## Pull Request Process

1. Create a feature branch: `git checkout -b feature/your-feature-name`
2. Make your changes
3. Test thoroughly
4. Commit with clear messages
5. Push to your fork
6. Open a Pull Request with a clear description

## Design Guidelines

- Maintain the elegant, feminine aesthetic
- Use the established color palette (pink/rose primary, amber accent)
- Follow the typography system (Playfair Display, Cormorant Garamond)
- Ensure responsive design for all screen sizes
- Test on multiple browsers

## Questions?

Open an issue for any questions or concerns.
